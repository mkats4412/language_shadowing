# Language Shadowing for YouTube 🎙️

Language Shadowing for YouTube は、自分の声を録音してすぐに再生（エコーバック）できる、学習用 Chrome 拡張機能です。
語学学習のシャドーイングや、マイクのテスト、スピーチの練習などに最適です。

![Language Shadowing for YouTube](icons/icon128.png)

## ✨ 主な機能

- **Push-to-Talk 録音＆再生**:
  指定したショートカットキー（またはポップアップ内のボタン）を押している間だけマイク音声を録音し、離した瞬間に録音した音声をそのまま再生します。
- **動画の自動一時停止・再開**:
  YouTubeなどの動画サイトで学習する際に非常に便利です。
  - **録音時一時停止**: 録音を開始した瞬間に、再生中の動画を自動的に一時停止します。
  - **再生後自動再開**: エコーの再生が終了した瞬間に、プログラムが止めた動画を自動で再開します。
- **ショートカットキーのカスタマイズ**:
  ポップアップから、自分の使いやすい任意のショートカットキー（修飾キー + 任意のキー）を設定できます（デフォルト: `Shift + Space`）。
- **多言語対応 (i18n)**:
  ブラウザの言語設定に応じて、自動で以下の言語に切り替わります。
  対応言語: 英語(デフォルト), 日本語, スペイン語, 韓国語, 中国語(簡体字・繁体字)
- **バックグラウンド録音対応**:
  最新の Chrome Extensions Manifest V3 の `Offscreen API` を利用し、タブを切り替えたりバックグラウンドに回っても安定して録音・再生が可能です。

## 📦 インストール方法（開発者モード）

この拡張機能は、Chrome ウェブストアを通さずにローカルから直接インストールして使用します。

1. このリポジトリをダウンロード（または `git clone`）し、任意のフォルダに解凍します。
2. Google Chrome を開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** のスイッチを **ON** にします。
4. 画面左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. 手順1で解凍した `chrome_echo` フォルダを選択します。
6. 拡張機能が一覧に追加されればインストール完了です！

## 🚀 使い方

1. **マイクの許可**: 初回起動時、拡張機能のアイコンをクリックしてポップアップを開き、マイクの使用を許可してください。
2. **ページを再読み込み**: 初回インストール時や拡張機能の更新時は、対象のWebページ（YouTubeなど）を必ずリロード（`F5` または `Ctrl+R`）してください。
3. **設定の調整**: ポップアップ画面で、動画の自動一時停止・再開のオン/オフや、お好みのショートカットキーを設定します。
4. **録音と再生**:
   - Webページ上で、設定したショートカットキー（例: `Shift + Space`）を **長押し** します。
   - 押している間は録音され、画面上（拡張機能アイコン）に `REC` と表示されます。
   - キーを **離す** と、すぐに録音した自分の声が再生されます（`PLAY` と表示されます）。

> [!TIP]
> **ハウリング防止について**
> 録音中にスピーカーの音がマイクに入るとハウリング（キーンという音）が発生する場合があります。快適に利用するため、**イヤホンやヘッドホンの使用を強く推奨します**。

## 🛠️ 技術スタック / アーキテクチャ

- **Manifest V3**: 最新のChrome拡張機能の仕様に完全準拠。
- **Offscreen API (`chrome.offscreen`)**: Service Worker では利用できない `getUserMedia` や `Audio` の再生を実現するため、オフスクリーンドキュメント内でメディアの録音・再生を処理しています。
- **Message Passing & Queueing**: バックグラウンドスクリプト (`background.js`) で Promise キューを利用し、非同期処理の競合（Race Condition）によるエラーを完全に防いでいます。
- **Vanilla JS & CSS**: フレームワークを使用せず、軽量で高速に動作します。UIにはグラスモルフィズムを取り入れたモダンなデザインを採用しています。
